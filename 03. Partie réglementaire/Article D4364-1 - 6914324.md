# Article D4364-1

Les prothésistes et orthésistes pour l'appareillage des personnes malades ou atteintes d'un handicap comprennent les
professions suivantes :

1° Les orthoprothésistes ;

2° Les podo-orthésistes ;

3° Les ocularistes ;

4° Les épithésistes ;

5° Les orthopédistes-orthésistes.
